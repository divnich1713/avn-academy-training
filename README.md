# 🛡️ AVN Academy Training Platform

**AVN Academy** — это высоконагруженная специализированная образовательная платформа для обучения, адаптивного тестирования и контроля успеваемости курсантов академии. Система построена по классической трехзвенной архитектуре с тактическим дизайном (Dark Mode UI) и автоматической интеграцией с внешними сервисами (Discord).

---

## 🚀 Технологический Стек

* **Frontend:** React 18, TypeScript, Vite, TailwindCSS, `shadcn/ui` (Radix UI), **Recharts** (визуализация аналитики), Lucide Icons.
* **Backend (FastAPI):** Python 3.11, FastAPI, SQLAlchemy (асинхронные запросы), Redis (кэширование сессий и ELO), Uvicorn.
* **Edge Functions (Deno):** Deno v2, TypeScript, прямой коннект к PostgreSQL (обработка транзакций вайтлиста, рапортов и запросов).
* **База данных:** PostgreSQL 17, миграции SQL.
* **Инфраструктура & DevOps:** Docker, Docker Compose, **Nginx** (кэширование, проксирование, gzip-сжатие, rate-limiting), автоматические скрипты бэкапов и развертывания (Paramiko SFTP).

---

## 👥 Ролевая Модель и Права Доступа

Платформа разделена на три уровня доступа:
1. **Курсант (Cadet):**
   - Доступ к памяткам, материалам и интерактивным флеш-картам.
   - Запуск адаптивных тестов по различным темам (с таймером и ограничениями).
   - Просмотр личной истории успеваемости и изменения рейтинга ELO.
   - Подача запросов на проведение лекций / практик и отправка рапортов на повышение (с загрузкой скриншотов-доказательств).
2. **Инструктор (Instructor):**
   - Управление очередью запросов курсантов (взятие в работу, одобрение, отклонение с комментариями).
   - Выставление оценок за лекции, практики и экзамены.
   - Просмотр списка курсантов, отслеживание оставшегося времени обучения (лимит 7 дней).
   - Выдача предупреждений (warnings) отстающим курсантам.
3. **Руководство Академии (Head / Deputy / Officers):**
   - Полное управление белым списком (`whitelist`): зачисление, редактирование, блокировка и удаление аккаунтов.
   - Массовые операции над группами курсантов (Bulk Operations).
   - Пакетное зачисление курсантов через импорт текстового CSV.
   - Доступ к сквозному журналу аудита действий администрации (Audit Log).
   - Просмотр расширенной аналитики успеваемости и мониторинга системы.

## ⚡ Ключевой Функционал Платформы

### 1. Интеллектуальное Тестирование на базе ELO 🧠
* **Адаптивный алгоритм:** Рейтинг ELO рассчитывается как для курсантов, так и для вопросов. Правильный ответ на сложный вопрос дает больше очков рейтинга; ошибка на легком вопросе сильнее снижает рейтинг курсанта.
* **Защита от списывания:** Ограничение времени на каждый вопрос, блокировка при переключении вкладок.
* **Детальный разбор попыток:** График времени ответа на каждый тип вопроса (choice, multichoice, matching, essay), визуализация ELO-прогресса, подробные объяснения к ответам.

### 2. Панель Инструктора и Группировка Запросов 📋
* **Управление очередью:** Вкладка запросов разделена на сегментированные вкладки по статусам с динамическими счетчиками («Новые запросы», «На рассмотрении», «Одобренные», «Отклоненные»).
* **Интеграция с Discord:** При создании запроса курсантом или его одобрении/отклонении инструктором, бот мгновенно отправляет форматированную карточку в соответствующий Discord-канал.

### 3. Журнал Аудита Действий Администрации (Audit Log) 🚨
* **Сквозное логирование:** Любое критическое действие администраторов (выставление оценки, создание/удаление курсанта, изменение настроек тестов, одобрение рапорта) записывается в таблицу логов аудита.
* **Интерфейс:** Поиск по имени инициатора, фильтрация по типу событий, пагинация и просмотр JSON-метаданных изменений в реальном времени.

### 4. Панель Аналитики (Analytics Dashboard) 📊
* **Визуализация метрик:** Общий объем попыток сдачи, средний балл по академии, процент успешности прохождения (Pass Rate).
* **Сложные вопросы:** Сводная таблица ТОП-10 самых трудных вопросов тестов на основе реальной статистики ответов (помогает выявлять опечатки и некорректные формулировки вопросов).
* **Рейтинг инструкторов:** Наглядный список самых активных проверяющих по числу действий в системе.

### 5. Пакетные Операции & Импорт CSV 👥
* **Массовые изменения (Bulk Actions):** Выбор нескольких курсантов чекбоксами для быстрой выдачи предупреждений, изменения звания/подразделения или удаления из базы.
* **CSV Импорт:** Панель быстрой вставки текстового CSV-формата для мгновенного зачисления взвода курсантов с автоматической предвалидацией ID.

### 6. Авто-Мониторинг Успеваемости (Guardrails) 🤖
* **Алерты отставания:** Платформа автоматически выявляет курсантов с **3 проваленными тестами подряд** (требуется помощь инструктора) и подсвечивает их красным цветом.
* **Контроль неактивности:** Автоматическое обнаружение курсантов, не проявлявших активность в системе более 7 дней с момента регистрации или последнего теста.

### 7. Звуковое сопровождение 🎵
* Заменена стандартная библиотека звуков на кастомный, чистый и приятный chime уведомлений (`social-media-logout-sound.mp3`), который воспроизводится в реальном времени при получении уведомлений о запросах и рапортах.

## 🛠️ Локальный запуск проекта

1. **Клонирование репозитория и настройка окружения:**
   - Скопируйте `.env.example` в `.env` и настройте ключи подключения, БД, JWT-секреты и Discord Webhook URL.
2. **Запуск через Docker Compose (рекомендуется):**
   ```bash
   docker-compose up --build -d
   ```
   *Это запустит Postgres, Redis, FastAPI backend, Deno Edge-сервер и мигратор.*
3. **Локальный запуск Frontend (разработка):**
   ```bash
   npm install
   npm run dev
   ```
4. **Сборка Frontend для Production:**
   ```bash
   npm run build
   ```
